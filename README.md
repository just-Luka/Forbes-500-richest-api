# TOP 500 Richest people

Package scraps data from Bloomberg and provides you list of real-time billionaires.

## Installation

>$ composer require m3duza/billionaires @dev

## Requirements
  - php ^7.3
  - guzzlehttp/guzzle: ^7.5

## Example

```
$repo = new BillionaireRepository()
$repo->all()
```
